Позиционированние

Позиционирование - это термин, означающий возможность с помощью CSS изменить стандартную логику потока для элемента.

То есть, мы можем, подвинуть элемент куда-нибудь, можем разместить один элемент сверху над остальными, можем вообще зафиксировать элемент в какой-то части экрана, чтоб он оставался там, даже если страница будет скроллится.

Пример 1

Использование свойства position: static; у каждого блока (значение по умолчанию)

.block-1
.block-2
.block-3

Пример кода CSS

.block { position: static; }

Пример 2

Использование свойства position: relative; + top + left у блока .block-2

.block-1
.block-2
.block-3

Пример кода CSS

.block-1 { position: relative; top: 50px; left: 50px; }

Пример 3

Использование свойства position: absolute; + top + left у блока .block-3 + position: relative; у родителя

.block-1
.block-2
.block-3

Пример кода CSS

.blocks-3 { position: relative; } .block-3 { position: absolute; top: 0px; left: 0px; }

Пример 4

Использование свойства position: fixed; + bottom + left у блока .block-1

.block-1
.block-2
.block-3

Пример кода CSS

.block-1 { position: fixed; bottom: 0px; left: 0px; }
